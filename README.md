# ChuckNorrisKotlin

### Purpose of this app is to show a knowing of simple concepts of Retrofit, RxJava and Android itself.

## Description

- Simple app where user can get random Chuck Norris joke through button click

## Stack:
- Kotlin + java
- Retrofit (2.9.0)
- RxJava (2.0.0)
